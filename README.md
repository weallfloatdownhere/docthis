# docthis
Opinionated yaml to makedown documentation maker in python.

---

## Installation

```bash
$ cd docthis/
$ pip3 install -r requirements.txt
```

## Usage

***python3 docthis.py generate `<outputdirectory>`***

*eg:*

```bash
$ p3 docthis.py generate .
```

## Dependencies

- [Jinja2-cli](https://pypi.org/project/jinja2-cli/)
